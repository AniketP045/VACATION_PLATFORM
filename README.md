# 🏨 Vacation Rental Platform – StayNest

A full-stack **Vacation Rental Web Application** that allows users to browse, book, and manage rental properties such as hotels, villas, and homestays. This project is built using **Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript** with secure authentication and real-time data handling.

---

## 📌 Project Overview

**StayNest** is an online vacation rental platform where:
- Users can explore properties
- View details with images & prices
- Book properties for a specific date
- Add to wishlist / favorites
- Admin can manage listings and users

This project is developed as a **college major project + full-stack practice system**.

---

## 🎯 Objectives

- To provide a smooth online vacation rental booking system  
- To implement full CRUD operations  
- To integrate authentication & authorization  
- To store and manage real data using MongoDB  
- To create a real-world full-stack project
